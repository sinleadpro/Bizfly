# Bizfly

## 功能

### 官網
- 測試機註冊 Quick Pass

### 結帳頁
- 顯示表單內容

### 前台
- 自動隱藏 `點我回後台管理`
- 前往後台按鈕
- 顯示 Shop Id, Theme Id

### 後台
- 後台 訂單/商品/顧客 不另開新分頁

## 開發 & 安裝
  - `git clone https://gitlab.sinlead.com/mickplayer/Bizfly`
  - Chrome `chrome://extensions/` 載入未封裝擴充功能
  - Firefox `about:debugging` 載入暫用附加元件
  - 前往商家前台，點擊擴充功能icon開啟
